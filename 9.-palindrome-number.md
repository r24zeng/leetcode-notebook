# 9. Palindrome Number

{% hint style="info" %}
Too easy.
{% endhint %}

```python
class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        if x < 0:
            return False
        
        x2 = 0
        temp = x
        while temp != 0:
            x2 = x2*10 + temp%10
            temp = temp//10
        if x2 == x:
            return True
        else:
            return False
```
